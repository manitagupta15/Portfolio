Nought and Crosses
A true classic Game. In this, Users can click on squares to place an alternating X or O. If a user places 3 X or O's horizontally, vertically or diagonally then the game let's them know that they've won! Keep track of whose turn it is. Players can only choose squares that haven't already been taken. Once a player has won allow them to reset the game. Keep track of the players scores. Alternate who goes first.

App Link : https://manitagupta15.github.io/naught-and-crosses/
